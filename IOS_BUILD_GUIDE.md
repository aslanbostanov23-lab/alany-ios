# 📱 Инструкция по сборке и компиляции iOS-приложения Alany

Мы полностью подготовили мобильное приложение для iOS (`alany-ios`), настроили необходимые системные разрешения и подготовили автоматическую сборку через GitHub.

---

## 🛠️ Что было сделано:
1. **Premium Полный Экран (Edge-to-Edge)**: В `App.js` мы убрали рамки `SafeAreaView`, сделали системный статус-бар прозрачным и позволили приложению растягиваться на весь экран. Это раскроет всю премиальность CSS-верстки вашей соцсети с учетом вырезов (челки/нотча) на iPhone.
2. **Мультимедиа и Круглые видео-заметки**: В свойства `<WebView>` добавлен параметр `allowsInlineMediaPlayback={true}`. Теперь круглые видео-заметки и плееры будут проигрываться прямо внутри чата, не раскрываясь на весь экран.
3. **Разрешения iOS (Privacy Permissions)**: В `app.json` добавлены обязательные ключи для доступа к камере, микрофону и галерее с описаниями на русском языке. Приложение успешно пройдет модерацию App Store и не будет вылетать при записи аудио- и видео-заметок.
4. **Сборочный конвейер GitHub Actions**: Создан файл автоматического билда `.github/workflows/build-ios.yml`, который компилирует нативный проект через Xcode на серверах GitHub.

---

## 🚀 Вариант 1. Автоматическая сборка через GitHub Actions (Бесплатно, для Симулятора)

Этот способ компилирует готовое приложение под симулятор iOS на macOS-серверах GitHub без покупки лицензии разработчика Apple.

### Как запустить сборку:
1. Загрузите папку `alany-ios` в ваш репозиторий на **GitHub** (или создайте для нее отдельный репозиторий).
2. Как только вы сделаете `git push` в ветку `main` или `master`, GitHub автоматически запустит сборочную линию.
3. Вы также можете запустить сборку вручную:
   * Зайдите во вкладку **Actions** в вашем репозитории на GitHub.
   * Выберите workflow **«Build iOS App (Simulator & Native Build)»** слева.
   * Нажмите **«Run workflow»**.
4. После успешного завершения сборки (около 10-15 минут) внизу страницы появится артефакт **`alany-ios-simulator-build`**, содержащий zip-архив с готовым приложением.

### Как запустить на Mac:
* Скачайте и разархивируйте `alany-simulator.app.zip`.
* Перетащите файл `alany-mobile.app` мышкой прямо в окно запущенного iOS-симулятора в Xcode. Приложение мгновенно установится!

## 📲 Вариант 2. Сборка для реальных iPhone и App Store (через EAS Build)

EAS Build — это официальный облачный сервис от создателей Expo, который собирает готовые `.ipa` файлы для реальных устройств или для публикации в App Store.

### Что вам понадобится:
* Аккаунт на [Expo.dev](https://expo.dev) (бесплатный).
* Платный аккаунт Apple Developer ($99/год) — требуется Apple для подписи кода реальных устройств.

### Пошаговый процесс:
1. Установите EAS CLI на вашем компьютере:
   ```bash
   npm install -g eas-cli
   ```
2. Войдите в ваш Expo-аккаунт:
   ```bash
   eas login
   ```
3. Запустите процесс настройки проекта (свяжет локальный код с вашим аккаунтом на Expo):
   ```bash
   eas project:init
   ```
4. Запустите сборку для iOS:
   ```bash
   eas build --platform ios
   ```
5. **EAS сделает всё за вас**:
   * Он попросит войти в ваш Apple Developer аккаунт.
   * Автоматически создаст все сборочные сертификаты и профили подготовки на серверах Apple.
   * Соберет готовый `.ipa` файл в облаке.
   * Предоставит QR-код, отсканировав который камерой iPhone, вы сможете мгновенно установить приложение на свой телефон!

## 💻 Вариант 3. Сборка локально на Mac

Если у вас есть Mac и установленный Xcode:
1. Перейдите в директорию `alany-ios` в терминале.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Сгенерируйте нативную папку `ios`:
   ```bash
   npx expo prebuild --platform ios
   ```
4. Откройте сгенерированный проект в Xcode:
   ```bash
   open ios/alany-mobile.xcworkspace
   ```
5. Подключите ваш iPhone по USB, выберите его в списке устройств Xcode и нажмите кнопку **«Play» (Run)** для запуска!
